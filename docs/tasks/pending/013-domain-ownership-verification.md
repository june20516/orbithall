## 작업 개요
도메인 소유권 검증을 통해 진짜 도메인 소유자만 사이트를 등록할 수 있도록 개선

## 현재 문제
- `sites.domain`에 UNIQUE 제약조건 있음
- 선착순으로 도메인을 차지할 수 있음 (악의적 사용 가능)
- 진짜 소유주가 나중에 등록하려면 막힘

## 작업 범위

### 포함
- `sites.domain`의 UNIQUE 제약조건 제거
- 도메인 소유권 검증 방식 선택 및 구현
  - DNS TXT 레코드 방식
  - 또는 HTML 파일 업로드 방식
- 검증된 도메인만 활성화 가능하도록 변경
- 미검증 도메인은 `is_verified` 플래그로 관리

### 제외
- 복잡한 검증 로직 (단순하게 유지)

## 예상 시간
2-3시간
